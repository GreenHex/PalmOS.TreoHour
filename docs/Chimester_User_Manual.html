A Palm Treo/ Centro Chime Application with KeyGuard and Popup Activation

User Manual

What is Chimester?

Chimester will remind you of the passage of time. It has several user configurable options and a built-in popup clock and keyguard to give you a unified view of the device status. The clock displays the alerts, sound, message, email, battery level, tasks and appointments status.

Features of Chimester

  • Unified, identical views for Clock, Popup and Keyguard.
  • Smooth, fast operation.
  • Cool tabbed user interface.
  • Uses a high quality font and not bitmap images for large clock display text.
  • Complete device status display on one screen - pending alerts, device volume status, new message counts, new email counts, pending tasks count, upcoming appointments, battery level and memory status. Additional gadgets for signal strength, roaming indicator, current carrier name, day-of-week and even the standard, boring, BT, signal and battery gadgets.
  • Use as a mini applications launcher.
  • Use system MIDI tones for chimes.
  • User-configurable keyguard unlock buttons.
  • Tiny PRC (~ 52 KB), NO hooking to 'hede' notifications.

Installing Chimester

HotSync or copy Chimester.PRC to your Treo phone. It will apprear in the Preferences Panel under "Other". It should be installed in the internal memory for proper functioning.

Uninstalling Chimester

Delete Chimester from the [Home]->[Menu]->[Delete] window.

Compatibility

Chimester should work as described with the Treo 600, Treo 650, Treo 700p, Treo 680, Treo 755p and the Palm Centro Smartphones.

Operation

Invoking Chimester from the Preferences Panel will display the Chimester Preferences form. The options are straightforward and self-explanatory.

Limitations of Chimester

  • Chimes may not be accurate to the second. Time is flexible, after all.

Known Bugs and Issues

  • New email counts are sometimes not accurate. Cause unknown.

Version Date Remarks
1.0 04-Jul-2006 Initial Release.
  • Chimester is derived from TreoHour, but is a fresh implementation. Many features of TreoHour have been dropped and Chimester offers smoother and more reliable operation.
1.1 05-Jul-2006
  • Added Chimester icon.
  • Cleaned-up button handling routine. More button options available, including side buttons.
1.3 09-Jul-2006
  • Added RSSI, roaming gadgets in addition to the standard BT, Signal and Battery gadgets to clock display.
1.4 09-Jul-2006
  • Added day-of-week and current carrier indicators.
  • Re-layout of clock screen.
1.5 10-Jul-2006
  • Signal strength indicator also works in CDMA devices.
  • Added option to disable unlock text in keyguard. Don't enable this if you are the forgetful kind.
  • Fixed minor bug that prevented clock display on manually enabling keyguard.
1.6 12-Jul-2006
  • Full one-handed operation for popup activation and keyguard deactivation (except for "Menu" key).
  • Center key popup activation and keguard de-activation is handled properly.
  • Signal strength value is correctly displayed in CDMA devices.
  • Re-layout and UI adjustments in clock display.
1.7 14-Jul-2006
  • Complete device memory status display in clock - Storage RAM, Dynamic Heap, DB Cache, Card.
1.8 15-Jul-2006
  • Added no-timeout option to popup. If selected, the clock display will be displayed till it is manually dismissed or auto-poweroff kicks-in.
